# Description

A Java implementation of graph traversal algortihms. Includes Breadth-First Search (BFS) and Depth-First Search (DFS).

# How-to-run

1. Clone the repository
	- git clone https://github.com/LoayHalawani/Graph-Traversal.git
	- cd Graph-Traversal

2. Compile & run
	- javac Graph.java
	- java Graph
